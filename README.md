# Python-Tkinter-Calculator-Application

A clean and interactive **GUI Calculator** built using **Python** and the **Tkinter library**.

This project demonstrates:
- GUI Development
- Event-Driven Programming
- Arithmetic Logic Implementation

---

## Features

✔ Addition  
✔ Subtraction  
✔ Multiplication  
✔ Division  
✔ Clear Button  
✔ Simple GUI Layout  

## Tech Stack

- **Language:** Python 3.14
- **Library:** Tkinter (Built-in)

## How to Run This Project from ZIP File

## Step 1: Download the ZIP File

- Click **Code**
- Click **Download ZIP**
- Save it to your computer

 ## Step 2: Extract the ZIP File

1. Go to your Downloads folder
2. Right-click on `PythonProject1.zip`
3. Click **Extract All**
4. Click **Extract**
Now a folder named `PythonProject1` will appear.

## Step 3: Open the Project

### Option A – Double Click Method
- Open the extracted folder
- Double-click `PythonProject1.py`

### Option B – Using PyCharm

1. Open PyCharm
2. Click **Open**
3. Select the extracted `PythonProject1` folder
4. Run `PythonProject1.py`
